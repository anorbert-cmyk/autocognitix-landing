## Executive Summary

This document outlines a comprehensive SEO (Search Engine Optimization), GEO (Generative Engine Optimization), and LLM (Large Language Model) strategy for AutoCognitix to dominate automotive diagnostics search across 10 major European markets.

**Key Strategy Pillars:**
- **GEO-First Approach:** Optimize for AI citations in ChatGPT, Perplexity, Google AI Overviews (25% of all searches by 2026)
- **LLM Entity Optimization:** Build knowledge graph presence with DTC codes, symptoms, and repair solutions
- **Localized Long-Tail Keywords:** Target high-intent "how to" and symptom-based searches in each language
- **Voice Search Optimization:** Capture "near me" queries and conversational repairs for in-car assistants
- **Multi-Layer Content:** Technical content for mechanics, beginner guides for DIY enthusiasts, price-based searches

**Market Prioritization:**
- **Tier 1 (Full Strategy):** Hungary (home), Germany, UK, France, Spain
- **Tier 2 (Targeted Strategy):** Italy, Poland, Netherlands, Romania, Czech Republic

## GEO (Generative Engine Optimization) - ALL MARKETS

### Core GEO Principles (Applicable to All Languages)

**1. Direct Answer First (40-60 words)**

Every DTC code page should start with:
```
P0401 [System] Error = [What it means in simple terms]. This is [how serious], 
typical in [vehicle models]. Fix: [1-2 solutions]. Cost: [€X-Y]. Time: [X hours].
```

**2. Fact Density (Every 150-200 words)**

Include verifiable facts:
- Repair cost ranges (average 2026 prices)
- Repair time estimates
- Which vehicles most affected
- Safety rating (safe/caution/dangerous)
- References to regulatory standards

**3. Source Attribution**

Every claim should have a source link:
```
According to [NHTSA/ADAC/TÜV] (link), P0401 affects approximately [X] vehicles annually...
```

**4. Platform-Specific Optimization**

| Platform | Optimization | Example |
|----------|-------------|---------|
| **ChatGPT** | Encyclopedic depth | "P0401 is an OBD2 code from the SAE J1979 standard..." |
| **Perplexity** | Recent data + examples | "2026 data shows P0401 affects 15% of 2019 Ford Focus units..." |
| **Google AI** | Existing top content | Optimize for #1-5 existing rankings first |
| **Claude** | Structured reasoning | Break down problem → cause → solution logically |

### Implementation Roadmap for GEO

**Phase 1 (Months 1-2):** Content Audit
- Map all 63 DTC codes to GEO requirements
- Identify which pages rank #1-5 on Google (these feed AI Overviews)
- Analyze competitor content structure

**Phase 2 (Months 2-4):** Content Restructuring
- Rewrite all DTC pages with 40-60 word direct answer at top
- Add fact density: cost/time/vehicles every 200 words
- Implement source attribution links

**Phase 3 (Months 4-6):** Schema & Entity Building
- Add FAQ Schema for all DTC codes
- Build Knowledge Graph entities (AutoCognitix + each DTC)
- Implement structured data for repair costs/times

**Phase 4 (Months 6+):** Active Outreach
- Monitor ChatGPT mentions of AutoCognitix
- Reach out to Perplexity when ranking content exists
- Track Google AI Overview inclusions

## LLM Optimization - ALL MARKETS

### Core LLM SEO Principles

**1. Entity Optimization (Knowledge Graph)**

Every page should define entities:
```json
{
  "mainEntity": {"@type": "Thing", "name": "P0401"},
  "relatedEntity": ["EGR System", "Engine Control", "Emissions"],
  "hasPart": ["EGR Valve", "MAF Sensor"],
  "hasRepairSolution": ["Clean EGR", "Replace Sensor"]
}
```

**2. FAQ Schema (LLM training data)**

LLMs learn from Q&A patterns:
```json
{
  "@type": "FAQPage",
  "mainEntity": [
    {
      "@type": "Question",
      "name": "What is P0401?",
      "acceptedAnswer": {"@type": "Answer", "text": "..."}
    }
  ]
}
```

**3. Breadcrumb Schema (Entity relationships)**

Show how topics connect:
```json
{
  "@type": "BreadcrumbList",
  "itemListElement": [
    {"name": "DTC Codes", "item": "..."},
    {"name": "P-Codes", "item": "..."},
    {"name": "P0401", "item": "..."}
  ]
}
```

### LLM Citation Strategy

**How to Get Cited by LLMs:**

1. **Be Authoritative on Specific Topics**
   - Pick 10-15 DTC codes to be THE expert on
   - Create 3,000+ word definitive guides
   - Include original data (repair costs by region, failure rates)

2. **Enable Easy Extraction**
   - Use Markdown headers (H1, H2, H3)
   - Use bullet lists (easier for LLM parsing)
   - Use tables for comparisons
   - Include bold keywords

3. **Original + Current Content**
   - LLMs prefer original research/data
   - Update repair cost data quarterly
   - Include 2026 regulatory changes
   - Monthly "DTC of the Month" spotlight

4. **Link to Authority**
   - Reference NHTSA, TÜV, ADAC, EU regulations
   - Partner with repair shops for cost data
   - Collaborate with mechanics for case studies
