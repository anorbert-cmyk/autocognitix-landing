2. Add/fix all meta descriptions (optimal: 150-160 chars, include keyword)
3. Fix H1 tag issues (remove duplicates, ensure primary keyword present)
4. Add alt text to all images

### Strategic Investments (4-8 hours)
1. Implement canonical URLs on all pages
2. Add hreflang tags linking HU ↔ EN versions
3. Add Article schema markup to blog posts
4. Expand blog articles to 2000-2800 words minimum
5. Add 5-8 internal links per blog article
6. Add 2-3 external authority links per article

### Long-term SEO Strategy (ongoing)
1. Build internal linking network across all blog articles
2. Target secondary and long-tail keywords
3. Implement breadcrumb schema
4. Add FAQ schema to relevant pages
5. Monitor keyword rankings and adjust content
