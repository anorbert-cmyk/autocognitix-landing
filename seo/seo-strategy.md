## 2. On-Page SEO Checklista

### 2.1 Generális Irányelvek

Minden oldalnak az alábbi minimum követelményeket teljesítenie kell:

#### Title Tag
- **Hosszúság:** 50-60 karakter (ideális)
- **Formátum:** `[Fő kulcsszó] - AutoCognitix | [Szekunder szó]`
- **Példák:**
  - `Autódiagnosztika AI-vel - AutoCognitix`
  - `DTC Kódszótár - Hibakódok Magyarázata - AutoCognitix`
  - `P0420 Hibakód - Causes & Solutions - AutoCognitix`

#### Meta Description
- **Hosszúság:** 150-160 karakter
- **Tartalmazzon:** CTA, kulcsszó, értékpropozíció
- **Formátum:** `[Problem] + [Solution by AutoCognitix] + [CTA]`
- **Példák:**
  - `Motorszámla lámpa? Fedezd fel a hibakódokat az AutoCognitix AI diagnosztikájával. Ingyenes analízis és alkatrészárak. Kezd most!`
  - `OBD2 hibakódok teljes listája magyarázattal. Tanulj meg minden DTC kódról és azok megoldásairól az AutoCognitix-szel.`

#### H1-H6 Hierarchia
- **H1:** 1 db/oldal, fő kulcsszót tartalmazza
- **H2:** 3-5 db, alszekciók
- **H3:** 10-15 db, részletes tartalom
- **H4-H6:** Szűkszáz, csak ha nagyon szükséges
- **Szabály:** Logikus sorrend, nem kihagyott szintek

**Példa Hierarchia (Blog Post):**
```
H1: Mit jelent a P0420 hibakód? Teljes útmutató
  H2: P0420 hibakód definíciója
    H3: Technikai megnevezés
    H3: Szűrő típusa
  H2: P0420 okái
    H3: Elhasználódott katalizátor
    H3: Lambda szonda meghibásodása
  H2: P0420 tünetei
  H2: P0420 javítása
    H3: DIY megoldások
    H3: Szerviz költségek
```

#### Image Alt Tagek
- **Minden képnek:** Leíró alt tag
- **Képletű:** `[Kontextus] - [Kulcsszó] - AutoCognitix`
- **Példák:**
  - `OBD2 diagnosztikai eszköz - hibakód olvasó adapter`
  - `P0420 hibakód diagram - katalizátor rendszer`
  - `AutoCognitix felhasználói felület - AI diagnosztika`
- **Ne:** Csak "kép", "diagram", "fotó"

#### Internal Linking
- **Stratégia:** Hub & Spoke (Pillar → Clusters)
- **Minimum:** 3-5 belső link/oldal
- **Anchor text:** Leíró, kulcsszót tartalmazó
- **Nyílás:** `<a href="/dtc/p0420" title="P0420 hibakód megoldása">P0420 hibakód</a>`

**Linking Stratégia:**
```
Landing Page (Hub)
  ├─→ DTC Szótár (Pillar)
  │    ├─→ P0420 Hibakód
  │    ├─→ P0171 Hibakód
  │    └─→ P0300 Hibakód
  ├─→ DIY Útmutatók (Pillar)
  │    ├─→ Gyújtógyertya Csere
  │    ├─→ Olajcsere
  │    └─→ Légszűrő Csere
  └─→ Blog (Pillar)
       ├─→ Motorszámla Lámpa
       ├─→ Lambda Szonda Hiba
       └─→ Katalizátor Hiba
```

#### URL Struktúra
- **Formátum:** `domain.hu/kategória/alkategória/cikk-slug`
- **Karakterek:** Kisbetűs, kötőjel, 3-5 szó
- **Kerülendő:** Számok, speciális karakterek, `?parameter`
- **Példák (HELYES):**
  - `autocognitix.hu/dtc/p0420-hibakod-megoldasa`
  - `autocognitix.hu/diy-utmutatoк/gyujtogyertya-csere`
  - `autocognitix.hu/blog/motorszamla-lampa-okai`
- **Példák (HELYTELEN):**
  - `autocognitix.hu/post?id=123&lang=hu`
  - `autocognitix.hu/BLOG/P0420_Hibakód`

#### Tartalomhossz
- **Landing Page:** 1,500-2,000 szó
- **Blog Post:** 1,800-2,500 szó
- **DIY Útmutató:** 2,000-3,000 szó
- **DTC Szótár (egyedi kód):** 800-1,200 szó
- **FAQ:** 1,200-1,800 szó
- **Minimum:** 800 szó (kivéve rövid tools/calculatorok)

#### Kulcsszó Sűrűség
- **Elsődleges kulcsszó:** 1-1,5%
- **LSI kulcsszavak:** 2-3%
- **Képmutató:** Természetes írás, nem erőltetett
- **Helyezés:** Title, H1, első 100 szó, utolsó 100 szó

## 3. Technikai SEO

### 3.1 Core Web Vitals Optimalizáció

Az AutoCognitix platformnak teljesítenie kell a Google Core Web Vitals követelményeit:

#### Largest Contentful Paint (LCP) - < 2.5 másodperc
- **Cél:** Landing page <1.5s, blog <2.0s, DTC page <2.0s
- **Optimalizáció:**
  - Kép optimalizálás (WebP formátum, srcset, lazy loading)
  - Kritikus CSS inline
  - Server-side rendering (SSR) frontend-hez
  - CDN (Railway + Cloudflare)
  - Font optimization (font-display: swap)

**Megvalósítás:**
```html
<!-- Kép optimalizálás -->
<picture>
  <source srcset="image.webp" type="image/webp">
  <img src="image.jpg" alt="..." loading="lazy">
</picture>

<!-- Font optimization -->
<link rel="preload" href="/fonts/inter.woff2" as="font" type="font/woff2" crossorigin>
<style>
  @font-face {
    font-display: swap;
  }
</style>
```

#### Cumulative Layout Shift (CLS) - < 0.1
- **Cél:** <0.05 ideális
- **Optimalizáció:**
  - Kép/videó dimensions megadása
  - Reklámok/beágyazott tartalom containerrel
  - Font loading előrejelzése
  - Animáció GPU-n futtatása

**Megvalósítás:**
```html
<!-- CLS elkerüléséhez -->
<img src="image.jpg" width="400" height="300" alt="...">
<div style="aspect-ratio: 16 / 9;">
  <iframe src="..."></iframe>
</div>
```

#### First Input Delay (FID) / Interaction to Next Paint (INP) - < 100ms
- **Cél:** <50ms ideális
- **Optimalizáció:**
  - JavaScript optimalizálás (splitting, tree-shaking)
  - React.memo, useDeferredValue hook
  - TanStack Query debouncing
  - Service Worker caching
  - Main thread deferred tasks

**Megvalósítás:**
```javascript
// Debounce keresés
const debouncedSearch = useCallback(
  debounce((q: string) => mutate(q), 300),
  [mutate]
);

// Code splitting
const DTC = lazy(() => import('./pages/DTC'));
const DIYGuides = lazy(() => import('./pages/DIYGuides'));
```

### 3.2 Mobile-First Responsive Design

- **Viewport meta tag:** `<meta name="viewport" content="width=device-width, initial-scale=1">`
- **Breakpoints:** 480px, 768px, 1024px, 1440px
- **Mobil-first CSS:** @media (min-width: ...) używaj
- **Touch target:** Minimum 44x44px gomb
- **Nem:** Plug-ins (Flash, Java) mobil verzióban

**Responsive Checklist:**
```
☐ Mobil display <3 másodperc
☐ Menü reszponzív (hamburger <768px)
☐ Táblázatok horizontálisan scrollozhatók
☐ Nyomtatott: CSS media print
☐ Tabbing sorrend logikus
☐ Zoom nem tiltott (user-scalable=yes)
```
